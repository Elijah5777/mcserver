FLINT HAVEN ECONOMY
// Server-only mod — no client install needed //
00
OPEN THE MAIN MENU

Everything starts here. You do not need the mod on your computer — only the server has it.

/ah /economy /menu

A chest menu opens with numbered options. Click an icon to go to that feature.

In-game help: /economy help
01
SELL AN ITEM ON THE AUCTION HOUSE
  • Open the main menu (/ah)
  • Click 2. Sell on Auction
  • Hold the item in your main hand (hotbar selected)
  • Click a gold nugget price ($100, $1K, $10K, etc.)
  • Click Confirm Listing (green block)

Or set a custom price first:

/auction sell 50000

Then confirm in the sell menu.

Cannot list: spawners. Max one stack (64) per listing.
02
REMOVE / CANCEL YOUR LISTING
  • Main menu → 3. My Listings
  • Click the item you want to remove
  • Item is returned to your inventory

Or use chat:

/auction cancel <listing-id>
03
BUY FROM THE AUCTION HOUSE
  • Main menu → 1. Auction House
  • Click any item in the top row to buy instantly
  • Use bottom buttons: pages, sort, claims, main menu

If your inventory is full, money still charges — claim later:

/auction claim
04
MONEY, PAY & LEADERBOARD
CommandWhat it does
/balanceYour balance
/balance SteveAnother player's balance
/pay Steve 5000Send money (instant, no confirm!)
/baltopTop 10 richest players
/pay is instant — double-check the name and amount before pressing Enter.
05
SERVER SHOP & ORDERS

Server Shop — fixed prices, infinite stock:

  • Main menu → 5. Server Shop
  • Click item = buy 1 | Shift-click = buy a stack
  • Full inventory? /shop claim

Orders — bulk trades:

/orders create diamond 64 1000

Hold matching items → /orders → click an order to deliver.

06
SELL TO SERVER (LOW PAYOUT)
  • Main menu → 7. Sell to Server
  • Hold item → click Confirm Sell

Much lower than auction prices — use for junk items only.

/sell
07
ADMIN COMMANDS OP ONLY

These require operator (OP) on the server. If you are on the server OP list, you can use them in-game or from console.

CommandWhat it does
/setmoney <player> <amount>Set exact balance
/addmoney <player> <amount>Add money
/removemoney <player> <amount>Remove money
/checkbalance <player>View balance (logged)
/ah admin remove-listing <id>Remove one listing
/ah admin clear-player <player>Clear all their listings
/shop reloadReload shop config.yml

Shop config file: config/flinthaven-economy/config.yml on the server.
